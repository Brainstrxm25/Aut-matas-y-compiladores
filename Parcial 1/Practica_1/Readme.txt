Práctica 1 – Análisis de Cadenas Regulares

En la carpeta “Práctica 1” se encuentra el material correspondiente al desarrollo de la primera práctica de la asignatura. Esta actividad está enfocada en el análisis de cadenas mediante el uso de lenguajes regulares y su implementación en el lenguaje de programación C++.

Dentro de la carpeta se incluyen los siguientes archivos:

Reporte_Practica1
Contiene el reporte completo donde se detalla el marco teórico, análisis de requisitos, modelo de flujo, desarrollo, implementación y conclusiones. En este documento se explica el fundamento teórico de los lenguajes formales, el análisis léxico y el uso de expresiones regulares.

cadenas_structura
Implementación en C++ que realiza el análisis de cadenas mediante validaciones manuales carácter por carácter. En esta versión se utilizan funciones auxiliares para clasificar cada token como número, palabra en mayúsculas, palabra en minúsculas, identificador, palabra reservada u operador.

cadenas_exprecionRegular
Implementación en C++ que realiza el análisis utilizando la biblioteca estándar <regex>. En esta versión se emplean expresiones regulares para identificar y clasificar los tokens, así como para contabilizar automáticamente las ocurrencias de cada categoría
